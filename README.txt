Toto prostředí slouží k práci s anotátorem detekovaných objektů.

Spusťte příkazový řádek v této složce.

V případě použití linux conda prostředí, spusťte:

    conda env create -f environment_linux.yml

Případně (Windows/nefungující conda) nainstalujte závislosti a knihovny pomocí

    pip install -r requirements.txt


Program se spouští v příkazové řádce pomocí

    python main.py

Podrobný návod k použití je k dispozici /docs/how_to_use.pdf
